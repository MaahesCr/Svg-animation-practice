Explanation

            1. An arbitrary triangle is created (in this case with an area less than 40,000)
            2. A random point X is generated inside the triangle 
            3. A segment is laid from point X to one of the vertices of the triangle
            4. A point is created in the middle of the resulting segment 
            5. Now it is considered that this is a new point X
            6. Steps 3 to 5 are repeated (5000 times in this example)

Пояснение

 
            1. Создается произвольный треугольник (в данном случае с площадью меньше 40000)
            2. Генерируется случайная точка X внутри треугольника 
            3. От точки X до одной из вершин треугольника прокладывается отрезок
            4. На середине полученного отрезка создается точка 
            5. Теперь считается что это новая точка X
            6. Повторяются шаги с 3 по 5 (в этом примере 5000 раз)
